QU'EST CE QUE GNLib ?

GNLib est une librarie créée par Guthen et Nogitsu.

COMMENT INSTALLER GNLib ?

- Téléchargement direct sur GitHub de la dernière version de GNLib, puis extraction du
dossier compressé obtenu vers le dossier addons de votre serveur ou jeu.
Bien que simpliste, cette technique est moins recommandée si vous souhaitez installer les mises à jour de
la librairie rapidement puisqu’il faudra répéter les étapes à chaque mise à jour.


- Installation sur le logiciel GitHub Desktop de GNLib dans le dossier addons.
Cette solution est plus pratique si vous souhaitez installer les dernières mises à jour rapidement.


- Téléchargement sur le Steam Workshop de GNLib, mais cette manière est plutôt
déconseillée parce que la librairie est souvent non mise à jour.
