GNLib.DrawRadialGradient

GNLib.DrawRectGradient( x, y, w, h, color1, color2, vertical )

DESCRIPTION :

Fonction permettant de dessiner un rectangle avec un dégradé vertical ou horizontal de deux couleurs.

ARGUMENTS :

• number x : Position en x
• number y : Position en y
• number w : Largeur du dégradé
• number h : Hauteur du dégradé
• Color color1 : Première couleur du dégradé
• Color color2 : Seconde couleur du dégradé
• boolean vertical=false : Si oui ou non le dégradé est horizontal ou vertical

EXEMPLE :

hook.Add( "HUDPaint", "GNLib:DrawRectGradient", function()
    GNLib.DrawRectGradient( 10, ScrH() - 50, 100, 40, GNLib.Colors.Clouds, GNLib.Colors.Amethyst )
end )
